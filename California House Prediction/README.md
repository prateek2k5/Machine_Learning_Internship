## 📖 Overview

This project was developed during **Week 1 of my Machine Learning with Python Internship**.

It predicts California housing prices using multiple regression algorithms and compares their performance. The project covers the complete machine learning workflow—from data preprocessing to model evaluation—and is deployed as an interactive **Streamlit web application**.

---

## ✨ Features

- Data preprocessing and feature scaling
- Exploratory Data Analysis (EDA)
- Multiple Regression Models:
  - Linear Regression
  - Polynomial Regression
  - Ridge Regression
  - Lasso Regression
- Performance evaluation using **RMSE** and **R² Score**
- Model comparison
- Interactive Streamlit deployment
- Prediction on unseen housing data

---

## 📊 Dataset

- **Dataset:** California Housing Dataset
- **Source:** Scikit-Learn
- **Target Variable:** Median House Value (`MedHouseVal`)


